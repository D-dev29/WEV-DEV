Project Name: "Interactive ToDo List"

Description: The "Interactive ToDo List" project is a simple yet practical web application that allows users to manage their tasks and stay organized. Built using HTML, CSS, and JavaScript, this project provides an interactive and user-friendly interface for adding, editing, and marking tasks as completed.
